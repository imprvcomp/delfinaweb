# Delfina Korn — Sitio web

Sitio web de la escritora Delfina Korn: obras, talleres, clínica de obra,
textos y contacto.

## Estructura

- `index.html` — sitio completo (HTML + CSS + JS, sin dependencias ni build).
- `assets/` — tapas de los libros y afiches de los talleres.

## Desarrollo

Abrí `index.html` en el navegador, o serví la carpeta:

```bash
python -m http.server 5500
```

Luego entrá a http://localhost:5500

## Edición

El contenido marcado con `<!-- EDITAR -->` en `index.html` usa textos
provisorios (bio, años/géneros de libros, prensa, redes) pendientes de
reemplazar por el contenido definitivo.
